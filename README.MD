# DogDex

DogDex is a responsive web application built with HTML, CSS, JavaScript, and Bootstrap.

The project consumes the Dog CEO API to display dog breeds, images, and breed information in a modern and interactive interface.

## Features

- Responsive UI
- Dynamic dog breed cards
- Search functionality
- Favorite system
- API integration with Dog CEO
- Modular JavaScript architecture

## Technologies

- HTML5
- CSS3
- JavaScript (ES Modules)
- Bootstrap 5
- Dog CEO API
